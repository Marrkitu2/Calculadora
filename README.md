# Calculadora
En este repositorio, podremos encontrar varias versiones de calculadora, algunas más trabajadas que otras.

Podremos encontrar lenguajes como:
- HTML
- CSS
- Java
- JavaScript
- Python

## Estructura del Repositorio

- `CalculadoraJava`: Contiene una calculadora científica desarrollada en Java utilizando Swing.
- `CalculadoraJavaScript`: Contiene una calculadora científica desarrollada en HTML, CSS y JavaScript.
- `CalculadoraPython`: Contiene una calculadora científica desarrollada en Python para la consola.

Cada carpeta tiene su propio README.md con información más específica sobre el proyecto.
