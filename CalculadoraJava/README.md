# Calculadora Científica en Java

Este proyecto contiene una calculadora científica desarrollada en Java utilizando Swing para la interfaz gráfica.

## Estructura del Proyecto

- `src/CalculadoraCientifica.java`: Archivo principal que contiene la implementación de la calculadora.

## Funcionalidades

- Operaciones básicas: suma, resta, multiplicación, división.
- Operaciones científicas: seno, coseno, tangente, raíz cuadrada, logaritmo, potencia, factorial.
- Modo científico: permite alternar entre el modo básico y el modo científico.

## Ejecución

Para ejecutar la calculadora, compila y ejecuta el archivo `CalculadoraCientifica.java`:

```sh
javac src/CalculadoraCientifica.java
java -cp src CalculadoraCientifica
