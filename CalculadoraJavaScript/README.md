# Calculadora Científica en JavaScript

Este proyecto contiene una calculadora científica desarrollada en HTML, CSS y JavaScript.

## Estructura del Proyecto

- `calculadora.html`: Archivo HTML principal.
- `styles/calculadora.css`: Archivo CSS para los estilos de la calculadora.
- `js/calculadora.js`: Archivo JavaScript que contiene la lógica de la calculadora.

## Funcionalidades

- Operaciones básicas: suma, resta, multiplicación, división.
- Operaciones científicas: seno, coseno, tangente, raíz cuadrada, logaritmo, potencia, factorial.
- Modo científico: permite alternar entre el modo básico y el modo científico.

## Ejecución

Para ejecutar la calculadora, abre el archivo `calculadora.html` en un navegador web.
