# Calculadora Científica en Python

Este proyecto contiene una calculadora científica desarrollada en Python para la consola.

## Estructura del Proyecto

- `calculadora.py`: Archivo principal que contiene la implementación de la calculadora.

## Funcionalidades

- Operaciones básicas: suma, resta, multiplicación, división.
- Operaciones científicas: seno, coseno, tangente, raíz cuadrada, logaritmo, potencia, factorial.

## Ejecución

Para ejecutar la calculadora, ejecuta el archivo `calculadora.py`:

```sh
python calculadora.py
